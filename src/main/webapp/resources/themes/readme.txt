
皮肤目录默认设置

皮肤目录页面类型目前共分为mc页面、登录页(login)、主页面(frame)、工作流审批(wf)

themes/xxx/css/mc-xx.css		mc框架核心css样式
themes/xxx/css/mc-ui-xx.css		mc框架使用的ui组件替换css样式
themes/xxx/js/					mc框架样式相关js
themes/xxx/images/				mc框架公用图片资源
themes/xxx/images/ui/			mc框架使用的ui组件替换图片资源

themes/xxx/login/				登录页(login)专用css、js、图片
themes/xxx/frame/				主页面(frame)专用css、js、图片
themes/xxx/wf/					工作流审批(wf)专用css、js、图片
